\### 🧠 核心细节速览



\#### 1️⃣ \*\*DOCTYPE 不可少\*\*

```html

<!DOCTYPE html>  <!-- 没有它，浏览器会进入"怪异模式"，样式错乱 -->

```



\#### 2️⃣ \*\*路径用正斜杠\*\* ✅

```html

<!-- ❌ Windows 风格 -->

<img src="images\\logo.png">



<!-- ✅ Web 标准（Linux/Mac/Win 通用） -->

<img src="images/logo.png">

```



\#### 3️⃣ \*\*图片必须有 alt\*\*

```html

<!-- ❌ 错误 -->

<img src="cat.jpg">



<!-- ✅ 正确：帮助视障用户 + SEO -->

<img src="cat.jpg" alt="一只橘猫在窗台上睡觉">

```



\#### 4️⃣ \*\*name vs id 的区别\*\*

```html

<input id="username" name="username">



<!-- 

&nbsp;  id = 给 CSS/JS 用的（页面唯一）

&nbsp;  name = 给服务器用的（表单提交时的字段名）

-->

```



\#### 5️⃣ \*\*GET vs POST\*\*

```

GET  → 搜索、筛选（数据在 URL 里，如 /search?q=手机）

POST → 登录、注册（数据隐藏，更安全）

```



\#### 6️⃣ \*\*语义化标签\*\*

```html

<!-- ❌ 全是 div -->

<div class="header">...</div>



<!-- ✅ 语义清晰，利于 SEO -->

<header>...</header>

<nav>...</nav>

<main>...</main>

<footer>...</footer>

```



---



\### 🔥 最容易犯的 3 个错误



| 错误 | 正确 |

|------|------|

| `<img src="pic.jpg">` | `<img src="pic.jpg" alt="描述">` |

| `<p>文字<div>内容</div></p>` | `<p>文字</p><div>内容</div>` |

| `class=box` | `class="box"` |

\## 📋 表单验证核心知识点总结



\### 🔹 一、基础验证属性（最简单）



| 属性 | 作用 | 示例 |

|------|------|------|

| `required` | 必填项 | `<input required>` |

| `type="email"` | 邮箱格式 | `<input type="email">` |

| `type="number"` | 数字输入 | `<input type="number">` |

| `minlength` | 最小长度 | `<input minlength="6">` |

| `maxlength` | 最大长度 | `<input maxlength="20">` |

| `min` / `max` | 数值范围 | `<input min="18" max="60">` |



\### 🔹 二、正则表达式验证（pattern）



```html

<!-- 手机号验证 -->

<input type="tel" pattern="^1\[3-9]\\d{9}$">



<!-- 身份证号 -->

<input type="text" pattern="^\\d{17}\[\\dXx]$">



<!-- 中文姓名（2-8个汉字） -->

<input type="text" pattern="^\[\\u4e00-\\u9fa5]{2,8}$">

```



\*\*常用正则符号：\*\*



| 符号 | 含义 | 示例 |

|------|------|------|

| `^` | 开头 | `^1` 以1开头 |

| `$` | 结尾 | `\\d$` 以数字结尾 |

| `\\d` | 数字 | `\\d{11}` 11位数字 |

| `{n}` | n次 | `\\d{6}` 6位 |

| `{n,m}` | n到m次 | `{2,8}` 2-8位 |

| `\[a-z]` | 小写字母 | `\[a-zA-Z]` 所有字母 |

| `\\|` | 或 | `A\\|B` A或B |



\### 🔹 三、自定义错误提示



```html

<!-- 方式1：title 属性 -->

<input pattern="..." title="这里是提示信息">



<!-- 方式2：oninvalid 事件 -->

<input required 

&nbsp;      oninvalid="this.setCustomValidity('自定义错误消息')"

&nbsp;      oninput="this.setCustomValidity('')">

```



\### 🔹 四、CSS 配合验证样式



```css

/\* 验证通过 - 绿色边框 \*/

input:valid {

&nbsp;   border-color: #2ecc71;

}



/\* 验证失败（且已输入内容）- 红色边框 \*/

input:invalid:not(:placeholder-shown) {

&nbsp;   border-color: #e74c3c;

}

```

&nbsp;          pattern="^\[a-zA-Z]\\w{5,15}$"

&nbsp;          title="字母开头，6-16位">

&nbsp;   

&nbsp;   <!-- 邮箱 -->

&nbsp;   

&nbsp;   <!-- 手机号 -->

&nbsp;          pattern="^1\[3-9]\\d{9}$">

&nbsp;   

&nbsp;   <!-- 密码：至少8位，含大小写+数字 -->

&nbsp;   <input type="password" required

&nbsp;          pattern="^(?=.\*\[a-z])(?=.\*\[A-Z])(?=.\*\\d).{8,}$"

&nbsp;          title="至少8位，包含大小写字母和数字">
